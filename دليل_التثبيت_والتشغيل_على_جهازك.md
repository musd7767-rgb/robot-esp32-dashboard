# دليل التثبيت والتشغيل على جهازك 🚀

## المتطلبات الأساسية:

قبل البدء، تأكد من تثبيت:

### 1. Node.js و npm
- اذهب إلى: https://nodejs.org/
- حمّل **LTS version** (الإصدار المستقر)
- ثبّت البرنامج
- تحقق من التثبيت:
  ```bash
  node --version
  npm --version
  ```

### 2. محرر النصوص (VS Code)
- اذهب إلى: https://code.visualstudio.com/
- حمّل وثبّت البرنامج
- هذا هو المكان الذي ستعدّل فيه الكود

### 3. Git (اختياري لكن مفيد)
- اذهب إلى: https://git-scm.com/
- حمّل وثبّت البرنامج

---

## خطوات التثبيت والتشغيل:

### الخطوة 1: استخراج الملفات

1. حمّل الملف `robot-esp32-dashboard.zip`
2. افتح File Explorer (مستكشف الملفات)
3. اذهب إلى المجلد الذي حمّلت فيه الملف
4. اضغط كليك يمين على الملف
5. اختر "Extract All" أو "استخراج الكل"
6. اختر مجلد لاستخراج الملفات فيه (مثل Desktop أو Documents)

**النتيجة:**
```
Desktop/
└── robot-esp32-dashboard/
    ├── client/
    ├── server/
    ├── package.json
    ├── vite.config.ts
    └── ... (ملفات أخرى)
```

---

### الخطوة 2: فتح المشروع في VS Code

1. افتح VS Code
2. اذهب إلى: File → Open Folder
3. اختر مجلد `robot-esp32-dashboard`
4. اضغط Select Folder

**ستشاهد:**
- القائمة اليسرى تعرض كل الملفات
- Terminal في الأسفل

---

### الخطوة 3: فتح Terminal في VS Code

1. اضغط `Ctrl + `` (الزر بجانب 1 في لوحة المفاتيح)
2. أو: View → Terminal
3. يجب أن تشاهد نافذة سوداء في الأسفل

---

### الخطوة 4: تثبيت المكتبات

في Terminal، اكتب:

```bash
npm install
```

أو إذا كنت تستخدم pnpm:

```bash
pnpm install
```

**ماذا يحدث:**
- سيحمّل كل المكتبات المطلوبة
- قد يستغرق 2-5 دقائق
- ستشاهد رسائل عديدة
- في النهاية ستشاهد: `added XXX packages`

---

### الخطوة 5: تشغيل خادم التطوير

في نفس Terminal، اكتب:

```bash
npm run dev
```

**ماذا يحدث:**
- الخادم سيبدأ التشغيل
- ستشاهد رسالة مثل:
  ```
  ➜  Local:   http://localhost:3000/
  ➜  Network: http://192.168.x.x:3000/
  ```

---

### الخطوة 6: فتح الموقع في المتصفح

1. افتح متصفحك (Chrome, Firefox, Edge)
2. اذهب إلى: `http://localhost:3000/`
3. يجب أن ترى Dashboard يعمل! 🎉

---

## 🐛 استكشاف الأخطاء:

### إذا لم يعمل شيء:

#### 1. تحقق من Terminal:
```
هل ترى رسالة خطأ؟
اقرأها بعناية
غالباً تخبرك بالمشكلة
```

#### 2. افتح Console في المتصفح:
```
اضغط F12
اذهب إلى Console
شاهد الأخطاء الحمراء
```

#### 3. أخطاء شائعة:

**❌ "npm: command not found"**
- Node.js لم يتم تثبيته
- أعد تثبيت Node.js

**❌ "Port 3000 is already in use"**
- البرنامج آخر يستخدم المنفذ
- اكتب: `npm run dev -- --port 3001`

**❌ "Cannot find module"**
- لم تشغّل `npm install`
- شغّل الأمر مرة أخرى

**❌ الصفحة بيضاء فارغة**
- افتح Console (F12)
- شاهد الأخطاء
- غالباً مشكلة في الكود
